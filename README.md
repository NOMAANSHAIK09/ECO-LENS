# ECO-LENS
AI INTEGRATED SUSTAINABLE DEVELOPMENT
🌱 EcoLens – AI Powered Sustainability Scanner

EcoLens is a web application that helps users understand the environmental impact of everyday products using AI.
Users can analyze products and get insights on eco-friendly usage, disposal methods, reuse ideas, and sustainability scores.

🔗 Live App:
https://eco-lens-9n1c.onrender.com/

🚀 Features

✨ AI Product Analysis
Analyze everyday products using AI powered sustainability insights.

♻️ Eco Score System
Each product receives an eco score indicating its environmental impact.

🧠 Smart Environmental Advice

Responsible usage

Safe disposal methods

Reuse ideas

Harm minimization tips

Eco-friendly alternatives

Biodegradable suggestions

🔐 User Authentication

User signup

Login system

Personalized experience

🌐 Web Based
Accessible from desktop and mobile browsers.

🧠 How It Works

1️⃣ User enters a product name
2️⃣ EcoLens sends the request to AI model (DeepSeek via OpenRouter)
3️⃣ AI analyzes the environmental impact
4️⃣ Results are returned in structured sustainability insights

⚙️ Installation (Run Locally)

Clone the repository

git clone https://github.com/YOUR_USERNAME/EcoLens.git
cd EcoLens

Install dependencies

pip install -r requirements.txt

Run the server

python backend.py

Open browser

http://127.0.0.1:5000
🔐 Environment Variables

Create a .env file

DEEPSEEK_API_KEY=your_api_key
DATABASE_URL=your_database_url
SECRET_KEY=your_secret_key
